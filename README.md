## Installation
Install the dependencies and devDependencies and start the server.
---Follow the instructions--- 

1.- git clone https://github.com/LucasKatz/proyecto-Cosmetologio-y-Yoga.git

2.- cd BackendProject

3.- npm install | yarn 

4.- npm run dev | yarn dev 

5.- Ready to edit!
